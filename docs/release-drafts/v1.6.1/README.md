# Scida v1.6.1 Release Draft Pack

This folder contains the concrete release-prep materials for `v1.6.1`.

Use this folder when preparing the actual `v1.6.1` release instead of starting from the generic templates.

## Included Files

- `checklist.md`
  - concrete release checklist copy for this version
- `commit-message.txt`
  - suggested release-prep commit message
- `tag-message.txt`
  - annotated tag message
- `github-release-body.md`
  - GitHub Release body draft for this version
- `release-day-commands.md`
  - shell command sequence for the actual release-prep day
- `release-day-runbook.md`
  - human step-by-step release operation card
- `staging-review.md`
  - pre-commit staging scope check for this version
- `release-closeout.md`
  - pre-push and post-release closeout card for this version
- `final-release-candidate-status.md`
  - one-page current release readiness status for this version

## Current Release Theme

`v1.6.1` is a focused polish release across the workflow surfaces added after `v1.4`, especially:

- database workspace hierarchy and interaction clarity
- controlled 24-hour test-time input
- simplified structured-data import and review
- Analysis structured-series naming consistency

## Important Notes

- This draft pack does not guarantee that packaging, signing, or notarization has actually passed.
- The final public release text must still reflect the real artifact trust state at release time.
