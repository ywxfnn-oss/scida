# Roadmap

Canonical roadmap: [docs/ROADMAP.md](docs/ROADMAP.md)

Scida's near-term plan now stays on a conservative `v1.6.x` line:

- `v1.6.2` Stability Patch
- `v1.6.3` Import Memory & High-Frequency Entry
- `v1.6.4` Analysis + Research Memory Polish

Product boundaries remain unchanged:

- local-first personal scientific data workspace
- not a team LIMS
- not a cloud ELN
- not an Origin replacement
- one structured data block = one X/Y curve
- analysis remains read-only
- export semantics remain stable
